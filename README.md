# 🍔 Swiggy Clone

A modern food delivery mobile application inspired by Swiggy, built using **React Native** and **TypeScript**.

> ⚠️ This project is developed for educational and learning purposes only.

## 📖 About Project

Swiggy Clone is a mobile application that replicates the core functionality of a food delivery platform. The app provides an intuitive food ordering experience with restaurant browsing, menu listing, cart management, and a responsive user interface. This project helped in gaining hands-on experience with mobile application development using React Native.

## ✨ Features

* 🍔 Restaurant listing and browsing
* 🍕 Food menu and category organization
* 🔍 Search functionality
* 🛒 Cart management system
* 📱 Responsive mobile UI
* ⚡ Smooth navigation and performance

## 🛠️ Technologies Used
